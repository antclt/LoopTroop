# Changelog

The canonical LoopTroop changelog, including current Unreleased entries, lives in [docs/changelog.md](docs/changelog.md#unreleased) and on the docs site at <https://www.looptroop.ovh/docs/changelog>.

Unreleased changes are tracked at the top of the canonical changelog until they are included in a versioned release.

This root file is kept as a discoverability pointer for repository visitors and tooling that expects a top-level changelog.
